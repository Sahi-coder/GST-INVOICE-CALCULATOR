# GST Invoice Calculator

A modern GST Invoice Calculator and Invoice Generator built using HTML, CSS, and JavaScript.

This tool helps businesses and freelancers generate GST-compliant invoices with automatic tax calculations for CGST, SGST, and IGST.

## Features

* Generate professional GST invoices
* Supports CGST + SGST (Intra-State)
* Supports IGST (Inter-State)
* Multiple invoice line items
* Automatic GST calculations
* Percentage and flat discounts
* Print-ready invoice format
* Save invoice as PDF using browser print
* Responsive design for desktop and mobile
* No login required
* No data stored
* Completely client-side application

## Supported GST Slabs

* 0%
* 3%
* 5%
* 12%
* 18%
* 28%

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Sahi-coder/GST-INVOICE-CALCULATOR.git
```

2. Open the project folder:

```bash
cd GST-INVOICE-CALCULATOR
```

3. Open `index.html` in any modern browser.

No installation or dependencies are required.

## How It Works

1. Enter business details.
2. Enter customer details.
3. Add products or services.
4. Select GST percentage.
5. Choose GST type:

   * CGST + SGST
   * IGST
6. Apply discount if needed.
7. Generate invoice.
8. Print or save as PDF.
